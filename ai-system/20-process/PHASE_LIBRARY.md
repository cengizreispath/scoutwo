# PHASE_LIBRARY.md
## Stable Phase Catalog

The phase library is stable.
The actual execution path is dynamic.

Core phases:
- Understand
- Frame
- Investigate
- Design
- Implement
- Verify
- Release Review
- Deploy Support
- Synthesize
- Recommend
- Learn

Optional specialized sub-phases:
- Diagnose
- Compare Options
- Critique
- Risk Review
- Migration Review
- Incident Review
- Documentation Review

Do not force every task through every phase.
