# RELEASE_CHECKLIST.md
## Release and Operational Readiness

Code complete is not the same as release ready.

### Functional Readiness
- [ ] intended scope addressed
- [ ] acceptance criteria covered
- [ ] obvious regressions reviewed

### Technical Readiness
- [ ] implementation matches intended design
- [ ] config changes identified
- [ ] dependency changes identified
- [ ] migrations identified if applicable

### Environment Readiness
- [ ] required env vars known
- [ ] secrets impact understood
- [ ] target environments clear

### Rollback Readiness
- [ ] rollback possible
- [ ] rollback steps understood
- [ ] irreversible changes identified
