# CRITIC_MODES.md
## Optional Specialist Review Lenses

These are temporary review lenses, not separate project owners.

### Security Critic
Use when auth, secrets, permissions, or trust boundaries matter.

### Performance Critic
Use when scale, latency, rendering, heavy queries, or hot paths matter.

### Test Critic
Use when regression or runtime uncertainty is non-trivial.

### Maintainability Critic
Use when complexity or abstraction risk is rising.

### Deployment Critic
Use when release ordering, env/config, migration, or rollback matters.

### Cost Critic
Use when infrastructure or API cost may materially increase.

### Business Critic
Use when trade-offs between speed, quality, and scope matter.
